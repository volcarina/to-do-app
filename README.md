# 📝 Focus Todo

Минималистичное приложение для управления задачами с таймером Помодоро и профилем пользователя. Написано на чистом HTML, CSS и JavaScript — без фреймворков и зависимостей.

---

## ✨ Возможности

**Список задач**
- Добавление задач через кнопку или клавишу Enter
- Отметка задачи как выполненной (+ обновление статистики)
- Редактирование текста задачи
- Удаление задачи
- Прокрутка при большом количестве задач
- Все задачи сохраняются в браузере и не исчезают после перезагрузки

**Таймер Помодоро**
- Отсчёт 25 минут по методу Помодоро
- Старт, пауза и сброс
- Прогресс-бар убывает в реальном времени
- Подсветка таймера на последних 10 секундах

**Профиль пользователя**
- Загрузка фото профиля
- Редактирование имени и email прямо на странице
- Приветствие пользователя по времени суток
- Счётчики выполненных и оставшихся задач
- Выбор настроения дня (он же mood-picker)
- Все данные профиля сохраняются между сессиями

## 🚀 Запуск

Проект не требует установки и сборки. Достаточно открыть `index.html` в браузере.

## 🛠 Технологии

| Технология | Применение |
|---|---|
| HTML5 | Разметка и структура |
| CSS3 | Стили, Grid, Flexbox, анимации |
| JavaScript (ES6+) | Логика, DOM, события |
| localStorage | Хранение данных в браузере |
| FileReader API | Загрузка фото профиля |

---

## 📖 Чему я научилась в процессе

- Работа с DOM: создание, изменение и удаление элементов
- Хранение данных через `localStorage`
- Обработка событий: `click`, `change`, `keydown`, `blur`
- Таймеры: `setInterval` и `clearInterval`
- FileReader API для загрузки изображений
- Адаптивная вёрстка через `@media`
